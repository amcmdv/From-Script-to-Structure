# Object-Oriented Programming Notes

## Object-Oriented Programming

OOP structures software around objects containing state and behavior.

---

## Class

A class defines a blueprint.

```javascript
class Greeter {}
```

---

## Constructor

A constructor initializes object state.

```javascript
constructor(name) {
    this.name = name;
}
```

---

## Methods

Methods define object behavior.

```javascript
greet() {
    return "Hello";
}
```

---

## Encapsulation

Encapsulation keeps related data and behavior together.

Benefits:

* cleaner design
* easier maintenance
* better scalability
