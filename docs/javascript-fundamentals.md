# JavaScript Fundamentals

## Variables

Variables store values for later use.

Example:

```javascript
let name = "Anthony";
```

---

## Functions

Functions encapsulate reusable logic.

```javascript
function greet(name) {
    return `Hello, ${name}`;
}
```

---

## Conditionals

Conditionals execute code based on conditions.

```javascript
if (name) {
    console.log(name);
}
```

---

## Loops

Loops repeat instructions.

```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```

---

## Error Handling

Use exceptions to manage invalid states.

```javascript
throw new Error("Invalid input");
```
