# OOP and JavaScript Notes

A simple restructure including refactoring of code into an introduction structured educational repository demonstrating JavaScript fundamentals and Object-Oriented Programming through progressively improved implementations of a greeting application.

---

## Overview

This repository explores multiple approaches to solving the same problem, beginning with simple procedural JavaScript and evolving into object-oriented design.

The goal is to demonstrate how software improves through abstraction, refactoring, and reuse.

---

## Topics Covered

### JavaScript Fundamentals

* Variables
* Functions
* Loops
* Conditionals
* Error Handling
* Regular Expressions

### OOP Concepts

* Objects
* Classes
* Constructors
* Methods
* Encapsulation
